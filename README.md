### Set up

`$ yarn`

### Start Application

`$ yarn start`

### Configuration

All constants are placed in `src/config/AppConst.js`

### Deployment

    $ git push origin heroku

## Environment Variable

    $ heroku config:set REACT_APP_BASE_URL=https://dev-textcampaign-api.herokuapp.com
